// SYSTEM_OVERVIEW

V3 Overview

Based on the issues with Version 2, Version 3 was fully developed within the 3D CAD model to simplify and improve the ball intake and shooting system. The Indexer was removed to reduce obstacles, and a direct ball path was designed.
  • Direct path width slightly less than ball diameter to maintain stability
  • Gate mechanism to temporarily stop balls before launch
  • Integrated system that is fast, smooth, and stable
// MECHANISM_A

Intake Mechanism V3

The Intake was modified to include four consecutive wheels that smoothly push the ball and gradually accelerate it toward the Shooter. This new design improves ball flow and prevents jamming.

Design Features
  • 4 consecutive wheels
  • Gradual ball pushing
  • Direct path without obstacles
  • Path width less than ball diameter
// MECHANISM_B

Gate Mechanism

A Gate mechanism was designed within the 3D model to temporarily stop the balls before launch until the robot reaches the appropriate launch position.

Functions
  • Temporary ball stopping
  • Launch timing control
  • Prevents balls before command
  • Quick opening at target location
// MECHANISM_C

Shooter Mechanism

Hood Control
  • Movable Hood with rear gears
  • Closed angle: 24° / Open angle: 50°
  • Servo Motor for precise control
Wheel Specifications
  • Diameter: 9 cm (radius 4.5 cm)
  • Material: High friction rubber
  • TPU ball for balance

- Linear speed at wheel: 21.19 m/s

Key Features
  • High friction ensures stable launch
  • High aiming accuracy
  • Fast and sequential shooting
  • Chain + Gear 18T/24T system
