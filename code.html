Limelight Shooter Tracking

In V1, we integrated the Limelight to track the target hub. The tracking algorithm calculated the offset (tx) and adjusted the drivetrain to align the shooter automatically before launching the ball.

// VisionShooter.java alignToTarget()
double tx = limelight.getTx();
double headingError = tx;
double steeringAdjust = Kp * headingError;

leftDrive.setPower(steeringAdjust);
rightDrive.setPower(-steeringAdjust);
