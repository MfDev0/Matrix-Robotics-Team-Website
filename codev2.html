// VISION_SYSTEM

Camera & Vision System

We used the Limelight camera for vision and target detection. We programmed it to recognize AprilTags and accurately determine distance. The system sends target coordinates to the controller to adjust turret angle and shooter speed.

// LimelightCamera.java
- Stores: targetX, targetY, targetArea
- hasTarget(): checks tv entry (1 = target detected)
- updateTargetData(): reads tx, ty, ta from network tables
- calculateDistance(): uses [TARGET_HEIGHT * FOCAL_LENGTH / targetArea]
Challenge: Vision latency in low-light conditions.
Solution: Improved lighting using integrated LEDs and increased the camera update rate to 50Hz.
